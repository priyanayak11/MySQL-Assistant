📊 MySQL Query Assistant
A simple AI-powered app that converts natural language questions into SQL queries and fetches results from a MySQL database.

🚀 Features
- Ask questions in plain English
- Automatically generates MySQL queries
- Executes queries and displays results
- Built with Streamlit for an interactive UI

🛠️ Tech Stack
- Python
- Streamlit
- MySQL
- LangChain + Google Gemini
- SQLAlchemy

💡 Example Queries
“How many Customers?”
“Total sales by category?”

🔐 Note
App is currently configured to run on local MySQL
Secrets are managed using st.secrets and excluded from GitHub
